# **App Name**: OrgView

## Core Features:

- Hierarchical Display: Display items in a tree-like, hierarchical format.
- Nesting: Support nesting items to multiple levels.
- Add New Nodes: Support dynamically adding of nodes
- Data Storage: Store data about the tree in memory. Save and restore automatically.

## Style Guidelines:

- Primary color: Subtle blue (#A0C4FF) to indicate the main branch.
- Background color: Light, desaturated off-white (#F0F0F2) to ensure content readability.
- Accent color: Light green (#B2F0B2) for nodes in lower levels.
- Body and headline font: 'Inter', a sans-serif font, should be used throughout, providing a modern, neutral and readable aesthetic.
- Use dashed lines to visually connect related items in the hierarchy.
- Simple plus icons for adding new child elements
- Use subtle animations, like smooth transitions or fade-ins, when expanding or collapsing the items in the hierarchy to give feedback and context to the user
